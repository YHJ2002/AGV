WareRover

It Takes Two to Tango: A Holistic Simulator for Joint Order Scheduling and Multi-Agent Path Finding in Robotic Warehouses

About

WareRover is an AGV (Automated Guided Vehicle) warehouse simulation that simulates multi-AGV order picking and delivery on a grid map, with configurable schedulers, path planners, and order generation strategies.

It provides closed-loop joint optimization (order scheduling coupled with MAPF), high-fidelity warehouse modeling, and resilience evaluation under stochastic failures.
